# PROBLEM STATEMENT
Write a Python program to check alpha character, whether vowel or consonant. And also display “it is number”, if user give any number.

## EXPECTED OUTPUT

### OUTPUT 1
    Enter a value: 3

    3 is a digit

### OUTPUT 2
     Enter a value: j

    j is a consonant

### OUTPUT 3
    Enter a value: A

    a is a vowel

### OUTPUT 4
    Enter a value: @

    @ is a special symbol
